# Prompt knihovna — naše zlatá pravidla

## 1. Vždy uveď kontext
- Jaký framework, jakou verzi
- Jaký stack (TS, JS, Tailwind, ...)
- Pokud upravuješ existující kód, **ukaž ho v promptu** (`@filename`)

## 2. Dej příklad výstupu
- "Mělo by to fungovat takto: `<Component prop={...} />`"
- Lepší než dlouhý popis

## 3. Dej omezení
- "Bez externích knihoven"
- "Maximálně 50 řádků"
- "Bez `any` typů"

## 4. Říkej co NECHCEŠ
- "Nepřidávej animace"
- "Neukládej do localStorage"
- AI ráda "vylepšuje" — omez ji

## 5. Pro komplex úkoly použij CHATu, ne TABu
- Tab = boilerplate
- Chat = logika
- Agent = multi-file, ale s předem napsaným plánem
